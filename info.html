INFORMAÇÕES

JavaScript (JS) é uma linguagem de programação usada principalmente 
para tornar páginas web interativas, permitindo animações, formulários 
e mapas dinâmicos. Originalmente usada no navegador, agora também é usada 
no servidor com Node.js. JS tornou páginas web mais dinâmicas, ao contrário 
das páginas estáticas de antes. Com o tempo, surgiram bibliotecas e frameworks
para facilitar seu uso. É uma linguagem interpretada, ou seja, o código 
é traduzido diretamente pelo navegador, ao contrário de outras linguagens 
que precisam ser compiladas antes.

HTML (Linguagem de Marcação de HiperTexto) é a base da web, responsável por estruturar o conteúdo.
Já o CSS define aparência e o JavaScript dá interatividade.
O termo hipertexto se refere aos links que conectam páginas entre si.
O HTML usa marcação com tags para indicar a função do conteúdo. Um elemento é formado por:
Tag de abertura 
 (marca onde começa).
Tag de fechamento 
 (marca onde termina).
Conteúdo (texto, imagens etc.).
Atributos (informações extras, como class="exemplo").

CSS (Cascading Style Sheets) é usado para estilizar páginas web, controlando coisas 
como cores, fontes e layout. Ele aplica estilos a elementos HTML, permitindo modificar a 
aparência da página sem alterar o conteúdo. Diferente de linguagens de programação, o CSS 
é uma linguagem de estilo, não de marcação.
